# 🩺 Diabetes Risk And Patient Health Prediction Dashboard

A simple and interactive web-based dashboard built with **HTML, CSS, and JavaScript** to visualize and predict diabetes risk based on patient health data.

## 🚀 Project Overview

This dashboard helps users understand key health indicators associated with diabetes and provides a simple prediction interface based on machine learning insights from the Pima Indians Diabetes Dataset.

The project focuses on important diabetes risk factors such as:

* 🩸 Glucose Level
* ⚖️ BMI (Body Mass Index)
* 👩 Age
* 👶 Pregnancies
* 🧬 Diabetes Pedigree Function
* 💉 Insulin
* ❤️ Blood Pressure
* 📏 Skin Thickness

## ✨ Features

* 📊 Interactive Diabetes Risk Dashboard
* 📈 Health Data Visualizations
* 🎯 Simple Diabetes Risk Prediction
* 📱 Responsive Design
* 🎨 Modern User Interface
* ⚡ Fast and Lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
  
## 📂 Dataset

The project is inspired by the **Pima Indians Diabetes Dataset**, a widely used healthcare dataset containing diagnostic measurements used to predict diabetes risk.

## 🎯 Purpose

The goal of this project is to demonstrate how healthcare data can be visualized and analyzed through a simple web application, making diabetes risk assessment more understandable and accessible.
